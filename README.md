# Shanti Yoga — Mindful Movement & Inner Balance (Landing Page)

Shanti Yoga is a modern, premium landing page concept for a yoga studio that focuses on calm structure, mindful progression, and beginner-friendly guidance. The website is designed to clearly communicate class offerings (in-studio + online), highlight instructors, show a weekly timetable, present pricing plans, and convert visitors through a free trial signup flow and newsletter subscription.

This project prioritizes clean typography, balanced spacing, and accessible UI patterns—delivering a smooth, responsive experience across desktop, tablet, and mobile devices.

---

## Live Demo
- Add your live link here: `https://strong-cascaron-9f59a4.netlify.app/`

## Project Goals
- Present a calm, premium yoga brand identity through layout and typography
- Help users quickly understand class formats, schedule, and pricing
- Build trust using instructor profiles and a clear program structure
- Drive conversions via a free trial lesson signup form and newsletter

---

## Core Sections (What’s Included)

### 1) Header + Navigation
- Desktop navigation for quick access to Teachers, Timetable, Lesson Types, and Contacts
- Mobile drawer menu with overlay, accessible controls (`aria-expanded`, `aria-controls`)
- Primary CTA button for trial signup

### 2) Hero (Brand Positioning)
- High-impact headline and supportive copy focused on mindful movement
- “Read More” style CTA for additional engagement
- Visual hero media with descriptive alt text for accessibility

### 3) Online Practice (Value Proposition)
- Content structured to explain online learning benefits:
  - live classes for feedback
  - recorded sessions for flexible schedules
  - guided progression for consistency
- Clear “Trial Lesson” CTA and supportive micro-copy

### 4) Lessons (Types of Yoga)
- 3-card lesson layout showcasing:
  - Yoga for Beginner
  - Regular Practice
  - Individual Practice
- Simple, visual-first section designed for quick scanning

### 5) Timetable / Schedule
- Weekly schedule grid/table format with:
  - day name
  - class title
  - time range
- Strong layout hierarchy for readability and conversion

### 6) Community Benefits
- Feature-style content blocks to communicate studio strengths:
  - Best Teachers
  - Aesthetics & calm environment
  - Personal attention
  - Beginner-friendly foundations
  - Options for different goals/levels

### 7) Pricing Plans
- 3-tier pricing section with clear value breakdown:
  - One-Time Visit
  - Group Lessons (featured “Most Popular”)
  - Individual Lessons (1:1)
- Bullet lists with icons for quick comparison
- CTA buttons designed for conversion-focused flow

### 8) Teachers (Instructor Trust Layer)
- Instructor cards with:
  - profile images
  - experience badges
  - role/specialization
  - short bio
  - category tags (Beginner, Meditation, Mobility, etc.)
- Designed to build credibility and encourage signups

### 9) Free Trial Signup Form
- Conversion-focused form section with:
  - Name, Email, Phone fields
  - Consent checkbox
  - Inline error placeholders and live message region (`aria-live`)
- Structure supports professional validation + submission logic in JavaScript

### 10) Instagram Gallery
- Grid-based section for social proof and brand lifestyle visuals
- Accessible links and descriptive image alt text

### 11) Footer (Professional Contact + Subscription)
- Brand summary and social links
- Contact details (email, phone, location)
- Quick navigation links + studio hours
- Newsletter subscription form with feedback message area
- Legal links placeholders (Privacy / Offer / Terms)
- Copyright year auto-update via JavaScript

---

## Tech Stack
- **HTML5** (semantic structure + accessible markup)
- **CSS3** (custom responsive styling)
- **Vanilla JavaScript** (navigation, validation-ready UI patterns, dynamic footer year)
- **Google Fonts**: Lora, Montserrat
- **Ionicons** for consistent UI icons
